# 선형화 칼만 필터
- 비선형 시스템을 하나의 기준점 주위에서 선형화 시켜서 얻은 선형 모델 A와 H를 사용한다
- __하지만__ 선형화 모델은 기준점 주위에서만 비선형시스템과 비슷한 역할을 하게되므로, __기준점 주위를 벗어나면 큰 오류가 발생__
